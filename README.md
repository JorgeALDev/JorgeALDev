## Sobre

Olá! Sou Jorge Araújo, desenvolvedor de software e concluinte do curso de Análise e Desenvolvimento de Sistemas pelo IFCE – Campus Tabuleiro do Norte. Todas as disciplinas da graduação já foram concluídas, restando apenas a colação de grau.

Minha principal experiência está no desenvolvimento de aplicações Mobile utilizando Kotlin, mas também atuo em projetos Web, APIs REST e bancos de dados relacionais. Ao longo da graduação participei de projetos de extensão, organização de eventos acadêmicos e do desenvolvimento de sistemas voltados para aplicações reais.

Atualmente atuo no desenvolvimento do **GeoSaúde**, uma aplicação construída com Kotlin Multiplatform para Android e iOS, destinada ao trabalho de agentes de endemias em campo, com suporte offline, sincronização de dados e integração com APIs.

Também colaboro no desenvolvimento do **HubCompras**, plataforma web voltada à gestão de compras coletivas, contribuindo com melhorias no frontend, correções e implementações, além de apoiar demandas específicas do backend.

Acredito que a melhor forma de aprender é construindo software. Busco constantemente evoluir em arquitetura de software, boas práticas, experiência do usuário e novas tecnologias, sempre procurando desenvolver soluções organizadas, escaláveis e de fácil manutenção.

## Experiência em Projetos

### GeoSaúde *(Projeto Privado)*

Aplicação desenvolvida em Kotlin Multiplatform para Android e iOS, destinada ao trabalho de agentes de endemias durante atividades em campo.

Sou responsável pelo desenvolvimento da maior parte da aplicação Mobile, incluindo implementação das telas, autenticação, persistência local de dados, sincronização offline, integração com APIs e evolução contínua do sistema.

O aplicativo já passou por testes realizados pela Prefeitura de Limoeiro do Norte (CE) e segue em desenvolvimento com base nos feedbacks coletados durante sua utilização.

**Tecnologias**

- Kotlin Multiplatform
- Compose Multiplatform
- REST APIs
- SQLite
- Android
- iOS
- Git
